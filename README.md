# Guts ⚔️

A react components library.

## Tools

-   **Storybook** : UI playground tool allows to build and document components
    in isolation.
-   **Rollup** : JS bundler, for packing and distributing the library into a
    single module.
-   **Jest** : Test runner for JS.
